# 🚕 Cab Booking System

## 📌 Problem Statement

Design a simple cab booking system that allows users to add cabs, book a cab based on availability, and view all cabs.

---

## ⚙️ Approach / Logic

* Implemented using **Java (OOP concepts)**
* Created a `Cab` class to represent each cab
* Used **ArrayList** to store cab details
* Menu-driven program with options:

  * Add Cab
  * Book Cab
  * Display Cabs
* Booking logic checks for **availability and location**

---

## 🧱 Technologies Used

* Java
* OOP (Class & Objects)
* ArrayList
* IntelliJ IDEA

---

## ▶️ Steps to Run

1. Open the project in IntelliJ IDEA (or any Java IDE)
2. Compile and run `Main.java`
3. Choose options from the menu:

   * Add Cab
   * Book Cab
   * Display Cab Details

---

## ✅ Features

* Add new cabs
* Book available cabs
* View all cab details
* Simple and easy-to-use menu interface
